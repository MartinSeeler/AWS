# s3

## Doc
* [s3 - AWS CLI](https://docs.aws.amazon.com/cli/latest/reference/s3/)

## Switch
* ls - list segment
* cp - copy
* rm - remove
* mb - make a bucket
* website
* mv - move or rename
* sync
* rb - remove bucket

## help
````bash
aws s3 help
````

## syntax
* aws s3 \<switch\>

## Examples
```bash
aws s3 ls --profile \<UserName\>
````
