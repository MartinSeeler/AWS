# s3

## Doc
* [s3 - AWS CLI](https://docs.aws.amazon.com/cli/latest/reference/s3/)
* [GET Bucket (List Objects) version 2](https://docs.aws.amazon.com/AmazonS3/latest/API/archive-v2-RESTBucketGET.html)

## Switch
* ls - list segment
* cp - copy
* rm - remove
* mb - make a bucket
* website
* mv - move or rename
* sync
* rb - remove bucket

## help
````bash
aws s3 help
````

````bash
aws s3 cp help
````

## syntax
* aws s3 \<switch\>

## Examples

aws s3 ls --profile \<UserName\>
