# ec2

## Switch

## Syntax

## help
````bash
aws ec2 run-instances help
````

## Examples
````bash
 aws ec2 run-instances --dry-run --image-id ami-020caff809d5a5307 --instance-type t2.micro --profile \<UserName\>
````
